# brgndy-react-metronome

React Metronome Library Component

## installing

```
npm install brgndy-react-metronome
```

## Usage

Import components

```javascript
import {
  MetronomeButton,
  Metronome,
  MetronomeProvider,
} from "brgndy-react-metronome";

export default function Component() {
  return (
    <MetronomeProvider>
      <Metronome />
      <MetronomeButton />
    </MetronomeProvider>
  );
}
```
